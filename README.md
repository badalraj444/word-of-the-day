# Word of the Day App

A full-stack application that delivers a new word daily with meaning and example usage.

## Tech Stack

- Frontend: React (Vite)
- Backend: Node.js + Express
- Database: PostgreSQL
- Version Control: Git + GitHub

## Features (MVP)

- Get today's word
- View meaning and example
- Admin can add words

## Project Structure

/client - React frontend  
/server - Express backend
